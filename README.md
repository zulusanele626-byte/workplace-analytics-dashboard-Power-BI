# workplace-analytics-dashboard-Power-BI
This project aims to analyze organisational and operational performance using the Superstore dataset. The goal is to transform raw transactional data into meaningful insights that support decision-making around sales performance, customer behavior, product efficiency, and delivery operations

## Objectives

Analyze organisational performance using sales, customer, product, and delivery data
Identify key revenue drivers and performance trends
Evaluate operational efficiency and customer behavior
Build an interactive Power BI dashboard for decision-making

## dataset used:
<a href="https://github.com/zulusanele626-byte/workplace-analytics-dashboard-Power-BI/blob/main/superstore_final_dataset%20(1).xlsx">data set</a>

## Ask Questions (KPIS)
What are the total sales, average order value, and customer count?
Which categories, products, and regions perform best?
How do customer segments contribute to sales performance?
What are the sales trends over time?
How efficient is delivery across different shipping modes?

## Project Insight

A small number of categories and products drive most of the revenue
Sales show seasonal trends with noticeable fluctuations over time
Certain regions and customer segments dominate overall performance
Delivery efficiency varies across shipping modes
The organisation has strong sales but uneven performance distribution

## Conclusion 

This project provides an interactive Power BI dashboard that converts raw sales data into actionable insights. It highlights key revenue drivers, customer behavior patterns, and operational efficiency gaps, enabling data-driven decision-making for improved organisational performance.
